# hubot-cheer

Feeling depressed?

See [`src/cheer.coffee`](src/cheer.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-cheer --save`

Then add **hubot-cheer** to your `external-scripts.json`:

```json
["hubot-cheer"]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
